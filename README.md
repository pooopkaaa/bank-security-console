# Пульт охраны банка

Пульт охраны банка

## Запуск

Для запуска блога у вас должен быть установлен Python 3.

- Скачайте код
- Установите зависимости командой

```bash
pip install -r requirements.txt
```

- Создайте файл `.env` виртуального окружения в папке `project`

```
HOST=
PORT=
NAME=
USER=
PASSWORD=
```

- Запустите скрипт командой

```bash
python3 main.py
```
